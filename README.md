# Publish a Podcast from Obsidian (in beta)
This plugin publishes audio recordings as podcast episodes, and documents as blog posts using the https://pouch.talk API.

## Private Communities
**Each Pouch** also offers a built-in community with user accounts, group chat, and private podcast feeds where listeners can gather.

## Steps
1. Get a Pouch subdomain (at https://pouch.talk/i/setup_domain.html)
2. Set up the Pouch (watch the  video)
3. Install the Obsidian plugin using BRAT


## Get a Pouch
You can request a Pouch subdomain at https://pouch.talk/i/setup_domain.html
